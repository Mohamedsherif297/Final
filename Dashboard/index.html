⚠️ Hand Gesture Control Setup:

  1. Make sure Pi is in MANUAL mode
  2. Open Terminal and run: 
    cd Dashboard/HandGestures_AI_Driven

  3. Show your hands to the webcam
  4. Watch the event log below for gesture commands

✋ How to use:

  • Left hand (fist + thumb): UP=forward, DOWN=backward, LEFT/RIGHT=turn
  • Right hand (fist + thumb): UP/DOWN=tilt, LEFT/RIGHT=pan
  • Commands will appear in the Event Log when gesture script is running
