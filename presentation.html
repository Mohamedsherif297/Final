Project Overview

The IoT Surveillance Car is a highly modular, secure, and responsive robotics platform powered by a Raspberry Pi.

It acts as a physical security node capable of remote piloting, live audio/video streaming, and autonomous safety responses.

System Architecture

Built on a decoupled, thread-safe architecture.

Hardware Abstraction

36 independent modules managing GPIO, PWM, Motors (L298N), and Pan/Tilt Servos via a centralized Singleton Manager.

Dual Networking

MQTT handles low-bandwidth telemetry and control, while WebSockets broadcast high-performance 30FPS JPEG frames.

Safety Layer

Multi-trigger emergency stops, watchdog heartbeats, and continuous ultrasonic obstacle filtering.

The Web Dashboard

Zero-Install Control
